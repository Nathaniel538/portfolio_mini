Chat Forum Application


It is a real time chat application capable of serving multiple user connections on a single session. Created using the most popular pthon web framework - Django, it is a diverse application with all the basic functionalities of a chat application mirroring a social media platform.

The primary or basic unit in the idea behind the application is a 'forum'. A forum is a room or a group where users can enter and chat away particularly regarding the topic the forum represents or whatever the user wishes to type.

The only requirement in entering a forum is that you know it's name(i.e. each forum has a unique name). Therefore all users on the platform are free to enter any forum given that they know it's name.

The foum and the user are both unique entities with their own identities(description, image, name, etc.).
And to implement all this, we used various modules provided by python that can be integrated into Django.
